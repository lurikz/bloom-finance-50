# FinControl Frontend

## Categorias com cores

- O seletor de cor em gradiente abre em um popover dedicado.
- O usuário pode clicar ou arrastar dentro do gradiente para atualizar a cor em tempo real.
- O seletor só fecha ao clicar fora do componente inteiro.
- A cor escolhida continua sendo usada nos gráficos do dashboard.
